# Applied Mathematics

Este repositorio reúne diferentes notebooks de **matemáticas aplicadas**, incluyendo análisis meteorológico, modelos físicos y otros experimentos numéricos.

## Contenido del repositorio

Actualmente incluye:

* **Predicción de temperatura** mediante ajuste polinómico usando datos de Open-Meteo.
* **Ley de enfriamiento de Newton**, con resolución numérica y visualización de la dinámica térmica.

## Objetivo del proyecto

El objetivo es explorar técnicas cuantitativas aplicadas a datos meteorológicos reales, desarrollar modelos sencillos de predicción y practicar análisis numérico y visualización.
